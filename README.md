# ЭлектроСчёт

PWA-калькулятор для расчёта смет на электромонтажные работы.

## Демо

👉 https://getto-dev.github.io/check-electro/

## Возможности

- 📋 200+ услуг в 24 категориях
- 💰 Автоматический расчёт сметы
- 💸 Учёт скидок
- 📄 Экспорт в HTML для печати
- 📥 Импорт/экспорт смет
- ✏️ Ручное добавление позиций
- 📖 **Прайс-лист для просмотра цен** (страница /price)
- 🌙 Тёмная тема
- 📱 Устанавливается как приложение (PWA)
- 🔌 Работает офлайн

## Категории услуг

- ⚡ Кабель по полу/стенам
- 🏠 Кабель по потолку
- 🔌 Монтаж провода
- 📡 СИП монтаж
- 🔮 Ретро проводка
- 🌐 Слаботочные сети
- 📦 Кабель-канал
- 🔧 Трубы
- 🔨 Штробление
- 📋 Лотки
- 🔩 Гильзы
- ⚙️ Заземление
- 🏗️ Земляные работы
- 🕳️ Отверстия
- 🔌 Подрозетники
- 📦 Распаячные коробки
- ⚡ Расключение
- 💡 Розетки/Выключатели
- 🌡️ Отопление/Аквасторож
- 💡 Освещение
- 🏠 Бытовая техника
- 🔌 Боксы/Щиты
- ⚡ Силовое оборудование
- 🔄 АВР

## Разработка

```bash
bun install
bun run dev
```

## Сборка

```bash
bun run build
```

## Данные

Прайс-лист хранится в `src/lib/catalog.ts`. Это единый источник данных для калькулятора и страницы прайс-листа.
